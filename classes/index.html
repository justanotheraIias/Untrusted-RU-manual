Классы

Каждый класс уникален, впрочем, одни навыки являются общими между классами, а другие - уникальными навыками класса.

Нейтральные классы имеют собственные цели и могут победить вне зависимости от успеха взлома.

Лидер операции

Вы являетесь лидером операции NETSEC - ваша роль заключается в координации атак и ликвидации потенциальных угроз для вашей операции. Хотя у вас есть навыки взлома, вам нужно сосредоточиться на других вопросах для успеха всей операции. Не попадитесь.

Этот класс уникален - только один игрок в каждой игре может иметь эту роль.

Этот класс гарантированно участвует в каждой игре.

Фракция: NETSEC

Шанс взлома узла: высокий

Условия победы: Взломать целевой узел. Если у нас получится, даже если нас арестуют - мы точно выберемся из тюрьмы, учитывая, кто наш клиент...
